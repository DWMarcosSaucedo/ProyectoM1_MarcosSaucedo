# Uso de Inteligencia Artificial

Durante el desarrollo del proyecto utilicé ChatGPT como herramienta de apoyo técnico.

La IA se utilizó principalmente para:

- Resolver dudas puntuales sobre HTML, CSS y JavaScript.
- Ayudar a identificar y corregir errores de código.
- Explicar conceptos del DOM y eventos.
- Sugerir mejoras en la organización del proyecto
