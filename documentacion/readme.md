# Generador de Paletas 

## Descripción

Este proyecto fue desarrollado como parte del Proyecto M1.

La aplicación permite generar paletas de colores aleatorias en formato HEX. El usuario puede seleccionar la cantidad de colores que desea visualizar y generar nuevas combinaciones mediante un botón.

Además, cada color puede copiarse al portapapeles haciendo clic sobre él.

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- GitHub Pages

---

## Funcionalidades

- Generación de paletas aleatorias.
- Selección de 6, 8 o 9 colores.
- Visualización del código HEX.
- Copiar el código HEX al portapapeles.
- Mensaje de confirmación al copiar.
- Diseño responsive.

---

## Cómo ejecutar el proyecto

1. Clonar o descargar el repositorio.

2. Abrir la carpeta **Desarrollo** en Visual Studio Code.

3. Ejecutar el archivo **index.html** utilizando Live Server.

---

## Decisiones técnicas

- Se utilizó JavaScript para generar colores aleatorios.
- El DOM se modifica dinámicamente mediante createElement().
- Se utilizó CSS Grid para organizar la paleta.
- La última paleta se guarda mediante localStorage.
